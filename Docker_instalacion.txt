# Guía de instalación

Docker en Linux: Ubuntu 20.04.6 LTS

Se emplearon comandos explicados en:
- Documentación oficial: https://docs.docker.com/engine/install/ubuntu/#prerequisites 
- Blog: https://www.nosolohacking.info/instalacion-de-docker-en-ubuntu/
- Videotutorial: https://www.youtube.com/watch?v=Q5YtjXoCfPs

1. Actualización

sudo apt-get update

2. Instalación de paquetes para permitir el uso de repositorio sobre https

sudo apt-get install apt-transport-https ca-certificates curl software-properties-common

apt-transport-https: 

3. Descargar una llave GPG oficial de docker

curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo apt-key add -

4. Verificar la llave

sudo apt-key fingerprint 0EBFCD88

5. Descargar la versión estable para x86_64/amd64

sudo add-apt-repository “deb [arch=amd64] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable”

6. Instalación de docker

sudo apt-get update
sudo apt-get install docker-ce docker-ce-cli containerd.io

7. Verificamos la instalación.

Sudo docker version

8. Evitar llamar a root.

sudo usermod -aG docker matilda
(matilda es el nombre de mi usuario)

9. Para verificar cerrar la sesión, volver a abrir y luego:

docker version
